First, fork the repo
Copy the HTTP link of this repository
Go to the dir where you want to clone then right-click and open the terminal type git clone https://github.com/VinayakBadole/blogging-Page.git
Then it will create the folder under which all files will be there open the folder with VScode then do your code from the top open terminal and write git add .
Afterward, commit the changes git commit -m 'initial commit' where u have to write what you have done in 3 words
At last push your code to your repo git push
Then come to your repo click on Pull Request then click on new pull request then write what changes you have made and create it
